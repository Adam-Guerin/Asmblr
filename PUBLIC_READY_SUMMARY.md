## 🚀 Quick Start for Public Users

### For New Users
```bash
# 1. Clone the repository
git clone https://github.com/your-username/asmblr.git
cd asmblr

# 2. Run the public setup script
python setup_public.py

# 3. Start the application
streamlit run app/ui.py

# 4. Open http://localhost:8501 and follow the onboarding!
```

### Demo Mode (Instant Exploration)
```bash
# Enable demo mode for quick testing
echo "DEMO_MODE=true" >> .env
echo "DEMO_TOPIC='AI-powered task management'" >> .env

# Start and explore immediately
streamlit run app/ui.py
```
